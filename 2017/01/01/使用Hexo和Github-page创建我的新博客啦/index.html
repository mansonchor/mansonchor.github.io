使用Hexo和Github page创建我的新博客

2017-01-01 21:52

我的第一个博客应该建立在2010年，那时还是用PHP和wordpress，自己买的空间和域名，用着最常见的LAMP方案

后面实在没有精力和时间，关键每年都忘记续费，断断续续坚持了3年，最后终于在欠费1个月后，被下线了，之前的博客内容也找不回来了

2017，新的开始

重拾博客，主要出于几方面原因吧：

  1. 新的一年，总得给自己找找新意，现在搭建一个博客比以前容易太多了，GitHub page老早就有，加上现在用的Hexo，真的只需要关注产出的内容，其它都基本是快速搭建
  2. 从前搞博客是为了练手，打基础，顺便写一些简单的技术文章。工作这么几年，感觉好多技术、实践、想法能总结和沉淀下来比能实现更加重要。到了这把年纪，注重规划、积累越来越重要
  3. 出于工作原因，负责自媒体相关业务，总想自己也长期能写点东西，接下地气。国内各个技术社区和平台虽然能直接获得用户，但是功能上总是限制了，作为程序员总是想自己能操控一切的，所以最终我选择了Hexo + Github page，看情况再同步到简书和segmentfault
  4. 今年给自己定个小目标，至少每2周写一篇博客和坚持每天看书15分钟


Let’s Start~~Fighting!!
